        방문 수
        사이트에 들어온 세션 수입니다. 한 사람이 여러 번 오면 여러 번 잡힐 수 있어요.
        사람 수
        GA4가 추정한 실제 방문자 수에 가깝습니다.
        CTR
        검색 결과에서 노출된 뒤 클릭된 비율입니다. 낮으면 제목/설명 개선 후보예요.
        평균 순위
        Google 검색에서 평균적으로 몇 번째 위치에 보였는지입니다. 숫자가 낮을수록 좋습니다.
        전환 신호율
        방문 수 대비 주문/상담 버튼 클릭 비율입니다. 실제 결제 전 관심도를 보는 지표입니다.
